🔗 Communication Entre Les Fichiers

Main.java → Menu.java → Controller → Service → DAO → SQL Server

📌 Exemple : Assignment Flow

1. Menu.java → L’utilisateur choisit : Créer Assignment

⬇

2. AssignmentController.java → Reçoit l’action utilisateur

⬇

3. AssignmentService.java → Vérifie la logique métier

⬇

4. AssignmentDAO.java → Interface contenant les méthodes SQL

Exemple : addAssignment(), getAllAssignments()

⬇

5. AssignmentDAOImpl.java → Implémente les vraies requêtes SQL

INSERT INTO Assignments ...

⬇

6. ConnectionDatabase.java → Ouvre connexion JDBC avec SQL Server

⬇

7. SQL Server → Sauvegarde les données dans la table Assignments


📌 Relation Interface ↔ Implémentation

AssignmentDAO.java = Interface

Contient uniquement les signatures des méthodes.

Exemple : 

void addAssignment(Assignment assignment);

⬇

AssignmentDAOImpl.java = Implémentation réelle

Contient le code SQL et JDBC. 

public class AssignmentDAOImpl
implements AssignmentDAO
